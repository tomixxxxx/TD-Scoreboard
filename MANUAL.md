# TD-Scoreboard ダッシュボード 利用マニュアル

## 1. このツールについて
このツールは、各店舗・スタッフの売上や客数データを可視化し、分析するためのダッシュボードです。
お手持ちのエクセルファイルを読み込むだけで、自動的にグラフやランキングが生成されます。

URL: https://tomixxxxx.github.io/TD-Scoreboard/

## 2. 安心してご利用いただくために（セキュリティについて）
このツールは、**セキュリティとプライバシーを最優先**に設計されています。

*   **データは外部に送信されません**:
    読み込んだエクセルファイルの内容は、すべて**お使いのパソコンやスマートフォンのブラウザ内でのみ**処理されます。
    外部のサーバーにデータがアップロードされたり、保存されたりすることは一切ありませんので、個人情報や売上データが流出する心配はありません。

*   **アクセスの制限**:
    URLを知っている第三者がアクセスできないよう、簡易的なパスワードロックをかけています。
    関係者のみでパスワードを共有してご利用ください。

## 3. 基本的な使い方

### 手順1: ログイン
1. URLにアクセスすると、パスワード入力画面が表示されます。
2. 管理者から共有されたパスワードを入力し、「ログイン」ボタンを押してください。
   * 一度ログインすると、しばらくの間は再入力不要です。

### 手順2: データの読み込み
1. 画面上部のエリアに、集計したいエクセルファイルをドラッグ＆ドロップするか、「通常ファイルを選択」ボタンからファイルを選んでください。
   * 複数の年度のファイルを一度に読み込むことも可能です（例: 2024年、2025年のファイルを同時選択）。
2. ファイルの内容が表示されたら、年度（2024年など）が正しいか確認し、「決定して読み込む」ボタンを押してください。

### 手順3: データの閲覧
画面上部にある **Main / Personal / Store** の3つのタブで表示内容を切り替えられます。

*   **Main（全体サマリ）**:
    *   全店舗の月別売上推移グラフ、月別客数推移グラフを確認できます。
    *   総売上・客単価・指名売上・商品売上の部門別 TOP3 ランキング（カードをクリックすると Personal タブの詳細ページへ移動）、および店舗別ランキングチャートが表示されます。
*   **Personal（個人分析）**:
    *   スタッフ個人の成績を分析できます。
    *   総売上・客単価・指名売上・商品売上のいずれかの指標で絞り込んだ TOP10 ランキングが表示されます。
    *   リストのスタッフをクリックすると、月ごとの売上・指名数・指名率の推移グラフが表示されます。
*   **Store（店舗別データ）**:
    *   店舗ごとの売上シェア（円グラフ）、月次売上推移（折れ線・棒グラフ）を確認できます。
    *   表示する店舗をボタンで選択／切り替えができます。

### 「データの確認が必要です」エリアについて
データ読み込み後、画面上部に黄色い警告エリアが表示される場合があります。

これは「エクセルデータの一部が正常に読み込めず、集計から除外された」ことをお知らせするものです。  
表示されている場合でも、**読み込み成功した残りのデータは正常に集計されています**のでご安心ください。

*   **主な除外理由**: 店舗コードが空欄・エラー値（`#N/A` など）になっているデータ行が対象です。エクセルの計算式の参照先が壊れている場合などに発生します。
*   **詳細の確認**: 「除外されたデータ (○件) を表示」をクリックすると、除外された行の一覧（ファイル名・行番号・理由）を確認できます。
*   **最小化**: エリア右上の ▲ ボタンをクリックすると、警告エリアを折りたたんで隠すことができます。

## 4. その他・注意事項
*   **推奨環境**: Google Chrome, Microsoft Edge, Safari などの最新ブラウザでご利用ください。
*   **スマートフォンでの利用**: スマホでも閲覧可能ですが、グラフや表が見づらい場合は画面を横向きにすることをおすすめします。
*   **データの更新**: データは保存されません。最新の状況を確認したい場合は、その都度最新のエクセルファイルを読み込んでください。
